# 📧 rassilOCHKA — Приложение для персонализированной рассылки писем

Графическое приложение для автоматической рассылки персональных писем через Gmail с гибкими шаблонами и поддержкой Excel-таблиц.

![Python](https://img.shields.io/badge/python-3.8+-blue.svg)
![PySide6](https://img.shields.io/badge/PySide6-6.0+-green.svg)
![License](https://img.shields.io/badge/license-MIT-blue.svg)

## 📌 Возможности

- **Безопасный вход** — мастер-пароль, учётные данные Gmail хранятся в зашифрованном файле `config.enc`
- **Гибкие шаблоны писем** — используйте метки: `*имя*`, `*дата*`, `*время*`, `*место*`, `*задание*`. Программа автоматически подставит данные из Excel
- **Поддержка родовых окончаний** — конструкция `слово1(слово2)` позволяет выбирать вариант в зависимости от пола получателя
- **Проверка данных** — перед генерацией писем проверяется наличие всех необходимых столбцов и заполненность ячеек
- **Предварительный просмотр** — все сгенерированные письма можно просмотреть перед отправкой
- **Интеллектуальная отправка** — случайная задержка между письмами для снижения риска блокировки
- **Сборка в exe** — готовое приложение можно скомпилировать для Windows

## 🛠️ Требования

- Python 3.8 или выше
- Gmail-аккаунт с включённой двухфакторной аутентификацией и **паролем приложения**
- Установленные зависимости (см. `requirements.txt`)

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/Chebupelinka/rassilOCHKA.git
cd rassilOCHKA
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Запуск приложения

```bash
python main.py
```

### 4. Первый запуск

При первом запуске программа попросит:
1. Ваш Gmail-адрес
2. Пароль приложения Gmail
3. Придумать мастер-пароль для входа в программу

Эти данные будут сохранены в зашифрованном виде в файле `config.enc`. При последующих запусках нужно будет ввести только мастер-пароль.

## 📦 Сборка в исполняемый файл (exe) для Windows

### 1. Установка PyInstaller

```bash
pip install pyinstaller
```

### 2. Выполнение сборки

```bash
pyinstaller --onefile --windowed --name "RassilOCHKA" main.py
```

- Флаг `--onefile` создаёт один exe-файл
- Флаг `--windowed` убирает консольное окно (для GUI)

Готовый файл появится в папке `dist`. При первом запуске он создаст `config.enc` рядом с собой.

> **Примечание:** Если вы используете антивирус, он может временно блокировать программу из-за отсутствия цифровой подписи. Это нормально — добавьте файл в исключения.

## 📖 Как пользоваться

### 1. Подготовка Excel-таблицы

В программе есть кнопка **«Создать шаблон Excel»** — сохраните пустой шаблон с правильными столбцами.

**Требуемые столбцы** (порядок не важен):
- `почта`
- `имя`
- `пол`
- `дата`
- `время`
- `место`
- `задание`

**Важно:**
- Все данные должны быть в текстовом формате (даты и время пишите как строки, чтобы избежать автоматического преобразования Excel)
- Столбец `пол` может содержать: `м`, `муж` или `ж`, `жен`, `женский` — для выбора альтернативных вариантов в шаблоне

### 2. Создание шаблона письма

1. Перейдите на вкладку **«Шаблон сообщения»**
2. Введите тему письма и текст
3. Используйте метки в звёздочках: `*имя*`, `*дата*`, `*время*`, `*место*`, `*задание*`. Программа заменит их на данные из таблицы

**Для учёта рода** используйте конструкцию `слово1(слово2)`:

```
Дорогой(Дорогая) *имя*!
```

Если в столбце `пол` указано женское значение, будет подставлено «Дорогая», иначе — «Дорогой».

4. Нажмите **«Сгенерировать письма»**. Если в таблице есть пустые ячейки для используемых меток, вы увидите сообщение с номерами строк
5. После успешной генерации вы перейдёте на вкладку **«Заготовки»**

### 3. Отправка писем

1. На вкладке **«Заготовки»** вы увидите список всех писем. Кликните на любое, чтобы просмотреть содержимое
2. Укажите диапазон задержки между письмами (в секундах). Случайная задержка поможет избежать спам-блокировки
3. Нажмите **«Начать рассылку»**. Приложение откроет окно с логом отправки, где будет виден статус каждого письма

Процесс отправки выполняется в фоновом потоке — интерфейс не зависает.

## 🧩 Особенности

| Особенность | Описание |
|-------------|----------|
| **Проверка недопустимых меток** | Если в шаблоне встретится метка, не входящая в список разрешённых (например, `*эээ*`), генерация остановится с указанием ошибки |
| **Безопасное хранение** | Мастер-пароль и данные отправителя хранятся в зашифрованном файле `config.enc` (с использованием `cryptography`) |
| **Поддержка русского языка** | Вся работа с текстом ведётся в UTF-8 |
| **Совместимость с exe** | Приложение корректно определяет путь к конфигу как в исходном коде, так и в скомпилированном виде |

## ⚠️ Важные примечания

### Пароль приложения Gmail
Обязательно создайте пароль приложения в настройках Google: [https://myaccount.google.com/apppasswords](https://myaccount.google.com/apppasswords)

**Обычный пароль от Gmail не подойдёт!**

### Лимиты отправки
Gmail ограничивает количество писем в день:
- Для обычных аккаунтов — до 500 писем
- Для Google Workspace — до 2000 писем

При больших объёмах рассылки используйте задержку между письмами.

### Антивирусы
При сборке exe некоторые антивирусы могут ложно срабатывать. Это связано с отсутствием цифровой подписи. Программа безопасна — вы можете добавить её в исключения.
