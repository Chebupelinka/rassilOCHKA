# 📧 Рассылка писем (rassilOCHKA)

Приложение для автоматической персонализированной рассылки писем через Gmail с гибкими шаблонами и поддержкой Excel-таблиц.

![Python](https://img.shields.io/badge/python-3.8+-blue.svg)
![PySide6](https://img.shields.io/badge/PySide6-6.0+-green.svg)
![License](https://img.shields.io/badge/license-MIT-blue.svg)

## 📌 Возможности

- **Безопасный вход** – мастер-пароль, учётные данные Gmail хранятся в зашифрованном файле `config.enc`.
- **Гибкие шаблоны писем** – используйте метки: `*имя*`, `*дата*`, `*время*`, `*место*`, `*задание*`. Программа автоматически подставит данные из Excel.
- **Поддержка родовых окончаний** – конструкция `слово1(слово2)` позволяет выбирать вариант в зависимости от пола получателя.
- **Проверка данных** – перед генерацией писем проверяется наличие всех необходимых столбцов и заполненность ячеек.
- **Предварительный просмотр** – все сгенерированные письма можно просмотреть и отредактировать перед отправкой.
- **Интеллектуальная отправка** – случайная задержка между письмами (от и до) для снижения риска блокировки.
- **Сборка в exe** – готовое приложение можно скомпилировать для Windows (см. инструкцию ниже).

## 🛠️ Требования

- Python 3.8 или выше
- Gmail аккаунт с включённой двухфакторной аутентификацией и созданным **паролем приложения**
- Установленные зависимости (см. `requirements.txt`)

## 🚀 Установка и запуск из исходников

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/Chebupelinka/rassilOCHKA.git
   cd rassilOCHKA
Установите зависимости:

bash
pip install -r requirements.txt
Запустите приложение:

bash
python main.py
При первом запуске программа попросит:

Ваш Gmail адрес

Пароль приложения Gmail

Придумать мастер-пароль для входа в программу

Эти данные будут сохранены в зашифрованном виде в файле config.enc.

При последующих запусках нужно будет ввести только мастер-пароль.

📦 Сборка в исполняемый файл (exe) для Windows
Установите PyInstaller:

bash
pip install pyinstaller
Выполните сборку:

bash
pyinstaller --onefile --windowed --name "RassilOCHKA" main.py
Флаг --onefile создаёт один exe-файл, --windowed убирает консольное окно (для GUI).

Готовый файл появится в папке dist. При первом запуске он создаст config.enc рядом с собой.

Примечание: Если вы используете антивирус, он может временно блокировать программу из-за отсутствия цифровой подписи. Это нормально, добавьте файл в исключения.

📖 Как пользоваться
1. Подготовка Excel-таблицы
В программе есть кнопка «Создать шаблон Excel» – сохраните пустой шаблон с правильными столбцами.

Столбцы (порядок не важен): почта, имя, пол, дата, время, место, задание.

Заполните строки. Все данные должны быть в текстовом формате (даты и время пишите как строки, чтобы избежать автоматического преобразования Excel).

Столбец «пол» может содержать: м, муж или ж, жен, женский – для выбора альтернативных вариантов в шаблоне.

2. Создание шаблона письма
Перейдите на вкладку «Шаблон сообщения».

Введите тему письма и текст.

Используйте метки в звёздочках: *имя*, *дата*, *время*, *место*, *задание*. Программа заменит их на данные из таблицы.

Для учёта рода используйте конструкцию слово1(слово2). Пример:

text
Дорогой(Дорогая) *имя*!
Если в столбце «пол» указано женское значение, будет подставлено Дорогая, иначе – Дорогой.

Нажмите «Сгенерировать письма». Если в таблице есть пустые ячейки для используемых меток, вы увидите сообщение с номерами строк.

После успешной генерации вы перейдёте на вкладку «Заготовки».

3. Отправка писем
На вкладке «Заготовки» вы увидите список всех писем. Кликните на любое, чтобы просмотреть содержимое.

Укажите диапазон задержки между письмами (в секундах). Случайная задержка поможет избежать спам-блокировки.

Нажмите «Начать рассылку». Приложение откроет окно с логом отправки, где будет виден статус каждого письма.

Процесс отправки выполняется в фоновом потоке – интерфейс не зависает.

🧩 Особенности
Проверка недопустимых меток – если в шаблоне встретится метка, не входящая в список разрешённых (например, *эээ*), генерация остановится с указанием ошибки.

Безопасное хранение – мастер-пароль и данные отправителя хранятся в зашифрованном файле config.enc (с использованием cryptography).

Поддержка русского языка – вся работа с текстом ведётся в UTF-8.

Совместимость с exe – приложение корректно определяет путь к конфигу как в исходном коде, так и в скомпилированном виде.

⚠️ Примечания
Пароль приложения Gmail – обязательно создайте его в настройках Google (https://myaccount.google.com/apppasswords). Обычный пароль от Gmail не подойдёт.

Лимиты отправки – Gmail ограничивает количество писем в день (для обычных аккаунтов – до 500). При больших объёмах рассылки используйте задержку.

Антивирусы – при сборке exe некоторые антивирусы могут ложно срабатывать. Это связано с отсутствием цифровой подписи. Программа безопасна, вы можете добавить её в исключения.
